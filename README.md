## 📌 Project Overview
Career Navigator is a web-based intelligent career guidance system designed to help users choose suitable career paths in the IT domain. It uses **Machine Learning, Artificial Intelligence, and Web Technologies** to provide personalized career recommendations and a structured roadmap.

---

## 🎯 Objective
- Provide personalized career guidance
- Analyze user skills, interests, and aptitude
- Generate AI-based career recommendations
- Offer structured career roadmap
- Improve decision-making using ML + AI

---

## ⚙️ Features
- 🔐 User Registration & Login System  
- 🧑‍💼 User Profile Management  
- 🧪 Multi-Phase Assessment (Aptitude, Psychometric, Skills)  
- 🤖 Machine Learning Based Career Prediction  
- 🗺️ Career Roadmap Generation  
- 💬 AI Chat Assistant (Gemini API)  
- 📊 Interactive Dashboard  

---

## 🧠 Technology Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL (XAMPP)  
- **Machine Learning:** Python, Flask  
- **AI Integration:** Gemini API  

---

## 🏗️ System Architecture
The system is designed using a multi-layer architecture:
- Presentation Layer (Frontend UI)
- Application Layer (PHP Backend)
- Machine Learning Layer (Python + Flask)
- Database Layer (MySQL)
- AI Layer (Gemini API)
